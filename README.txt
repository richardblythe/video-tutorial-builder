Create a customized video tutorial collection for your wordpress admin.

== Description ==

This plugin was developed by a need to me to offer embedded website tutorials for my clients.  Not only does Video Tutorial Builder give clients easy access to helpful videos but I can also give my client the specific videos that are only relevant to them.

== Installation ==

1. Upload `vtb.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add video individually or import an entire playlist from YouTube or Vimeo

== Screenshots ==

1. The List screen allows users to view the entire list of imported videos.
2. The Player screen allows users to watch videos as well as select previous or next videos in the list.
3. The Edit screen allows administrative users to add new videos to the list.
4. The Settings screen allows administrative users to change general settings of the plugin.
5. The Import screen allows administrative users to import a collection of videos from a playlist or an exported .json file
6. The Export screen allows administrative users to export the settings and videos of the plugin.
